# Galeria de fotos

Essa é uma aplicação web, desenvolvida em React, que exibe uma galeria de fotos. Ela permite que os usuários visualizem, filtrem e pesquisem imagens com base em palavras-chave

## Tecnologias utilizadas

- React + Vite
- JavaScript
- CSS

## Como executar?

Para rodar o projeto localmente você deve possuir no [Node.js](https://nodejs.org/pt) instalado na sua máquina como pré-requisito.

1. Clone o repositório em sua máquina: ```git clone https://github.com/isadoracaarvalho/galeria-de-fotos.git```
2. Navegue ate o diretório:  ```cd galeria-de-fotos```
3. Instale as dependências: ```npm install```
4. Rodar o projeto localmente: ```npm run dev```
5. O projeto estara disponível na porta: http://localhost:5173/

ou acessar o seguinte link: [https://galeria-de-fotos-sepia-xi.vercel.app/](https://galeria-de-fotos-sepia-xi.vercel.app/)

## Créditos

As imagens exibidas nessa aplicação são fornecidas pela API [Pexels](https://www.pexels.com/)
